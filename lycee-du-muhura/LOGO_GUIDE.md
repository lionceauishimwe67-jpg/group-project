# Uburyo Bworoshye Bwo Shyiramo Ifoto ya Logo

## Intambwe Ngufi (3 Steps)

### 1. Fungura Folda
```
public/
```

### 2. Shyiramo Ifoto
- Kanda **COPY** kuri foto ya logo ushaka
- Kanda **PASTE** muri folda `public/`
- Hindura izina rihite rite **logo.png**

### 3. Reba Result
- Fungura browser kuri: `http://localhost:3000`
- Logo izagaragara hejuru mu kanyuguto (corner)

---

## Ibindi Bishoboka

### Niba ushaka gukoresha izina ryihariye:

**Hindura mu file:** `src/components/Logo.js`

```jsx
<img 
  src="/Lycée Saint Alexandre Sauli emblem.png"  // <-- Hindura hano
  alt="Lycée Saint Alexandre Sauli TVET - MUHURA"
  className="logo-image"
/>
```

## Aho Foto Igaragara

| Aho | Uburyo | Ingano |
|-----|--------|--------|
| Header | Hejuru hasance | 80px height |
| Mobile | Hejuru hasance | 60px height |

---

## Niba Ibibazo Byabaye

**Foto ntiyagaragara?**
1. Reka server (Ctrl+C)
2. Wonge ukomize: `npm start`
3. F5 (Refresh) kuri browser

**Foto nini cyane?**
- Bika neza: 200x200 pixels
- Cyangwa 300x300 pixels
- Format: PNG cyangwa JPG

---

## Byoroshye Kurushaho

Waba ushaka **DRAG & DROP** gusa?
1. Fungura folda `public/` muri Windows Explorer
2. Fata foto ushaka
3. Kuyogeza (drag) ukoreshe muri folda
4. Byihishe!
